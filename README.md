# 学术论文标题错误分类实验

## 实验背景

这是老师上学期课程上布置的实验，之前未来得及记录完整版。因为这学期复现了一下，做一下记录，希望可以帮到大家（代码还是vibe coding，readme文件也是由大模型结合我们的代码以及实验结果生成）🤣🤣

CiteSeer 是一个面向计算机与信息科学领域的学术论文搜索引擎与数字图书馆。在实际运行过程中，系统从 PDF 或网页中自动提取的论文标题经常存在错误（例如我们前面实验使用过的grobid工具），与原始标题不一致。例如，正确标题 "The Social Life of Routers" 被错误提取为 "Call for Papers......41 Fragments......42"。因此，构建能够自动识别错误标题的分类模型，对提升 CiteSeer 的数据质量具有重要意义。  
（正确理解这一段话可以帮助你更好地看懂测试集的表格）

## 实验方法

### 任务一：朴素贝叶斯分类器

该方法基于 TF-IDF 特征构建朴素贝叶斯分类器，具体流程如下：

- 使用 TF-IDF 对标题文本进行向量化，设置最大特征数为 20000，采用 1-3 元词袋模型，并去除英文停用词。
- 使用 MultinomialNB 作为分类器，通过设置类别权重来平衡正负样本。
- 在测试集上进行预测，输出分类结果并计算评估指标。
- 对 TF-IDF 特征进行 t-SNE 降维，可视化正负样本在特征空间中的分布。

### 任务二：Word2Vec + SVM

该方法结合词向量与 SVM 分类器，具体流程如下：

- 使用 Word2Vec 在训练集上训练词向量模型，向量维度设为 300，窗口大小为 8，最小词频为 3。
- 对每条标题提取以下特征：
  - 词向量的均值、最大值和最小值
  - 文本长度、词数、大写字母比例、数字比例、标点密度等统计特征
  - 特定关键词检测（如 "call for paper"、"abstract" 等）
  - TF-IDF 加权的词向量特征
- 特征拼接后进行标准化处理。
- 使用 LinearSVM 进行分类，通过网格搜索优化正则化参数 C 和类别权重。
- 对 Word2Vec 文档向量进行 t-SNE 可视化。

### 任务三：BERT 微调分类器

相较于未微调的版本，这版正确率有所提升。

该微调方法基于预训练语言模型 BERT 进行端到端的微调分类，具体流程如下：

- 使用 `bert-base-uncased` 预训练模型作为基础模型。
- 在 BERT 模型顶部添加分类层，输出维度为 2（正确标题 / 错误标题）。
- 按 8:2 的比例将训练集划分为训练集和验证集，保证类别分布一致。
- 使用 AdamW 优化器，学习率设为 2e-5，权重衰减为 0.01，训练 5 个 epoch。
- 在验证集上监控 F1 分数，保存最佳模型并在测试集上评估。
- 提取 12 层 Transformer 的 [CLS] 向量，分别进行 t-SNE 可视化。

## 实验结果

以下为三种模型在 1000 条测试集上的分类性能对比。

| 评估指标 | 朴素贝叶斯 | Word2Vec + SVM | BERT 微调 |
|----------|------------|----------------|------------|
| 准确率 | 0.7602 | 0.8094 | 0.8914 |
| F1 分数 | 0.7976 | 0.8339 | 0.9083 |
| 精确率 | 0.7840 | 0.8460 | 0.8929 |
| 召回率 | 0.8116 | 0.8222 | 0.9243 |

实验结果表明：

- 朴素贝叶斯方法在简单 TF-IDF 特征下表现一般，适合作为基线模型。
- Word2Vec + SVM 结合词向量与手工特征后，性能明显优于朴素贝叶斯。
- BERT 微调方法在所有评估指标上均取得最佳结果，尤其在召回率和 F1 分数上提升显著，表明预训练语言模型对标题语义的建模能力更强。
